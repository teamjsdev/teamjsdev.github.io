# Personal Website

Sitio web personal utilizado como presentación profesional y portfolio.

El foco principal del perfil se encuentra en el desarrollo de aplicaciones Android nativas, con experiencia en apps publicadas y mantenidas en Google Play.

La web está implementada con HTML simple, priorizando claridad, accesibilidad y funcionalidad por sobre complejidad técnica, ya que no representa el área principal de especialización.

## Código y repositorios

La mayoría de los proyectos Android se encuentran en repositorios privados debido al uso de cifrado, manejo de datos sensibles y criterios de seguridad.

Puedo compartir ejemplos de código, explicar decisiones técnicas o realizar una revisión técnica bajo solicitud.
